# Zentro

## Web API Server written in NestJS

## Description



## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deployment

[deployment documentation](https://docs.nestjs.com/deployment) 

Cloud-based platform to deploy your NestJS application - [Mau](https://mau.nestjs.com) official platform for deploying NestJS applications on AWS. 

```bash
$ npm install -g mau
$ mau deploy
```
